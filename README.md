# Github-information

### Termux Install
```
$ apt update && apt upgrade
$ pkg install git
$ pkg install python
$ git clone https://github.com/LeON101-coder/Github-Information
$ cd Github-Information
$ python github.py
```
### Linux Install
```
# sudo install git
# sudo install python
# git clone https://github.com/LeON101-coder/Github-Information
# cd Github-Information
# python github.com
```
